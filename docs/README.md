## Project Closed!

You can still use your rotation that you purchased 3/4/5/10 years ago without any restrictions.  
However, **DarhangeR Rotations** has moved to the **UNIVER Project** platform with new rotations, which will need to be purchased separately.

<br>

The authorization server will remain active until **May 2026**, so that all long-time users can log into the game, see the notification, and make a decision: move to the new platform, download, and continue using the product they purchased earlier.

<br>

On **May 1, 2026**, the authorization server will be shut down. On the page of the current project, there will be a button for downloading the software **without restrictions**.

<br>

🙏 I want to sincerely thank all of you for the long and productive **5 years of this project’s existence**. Thanks to your support and trust, it was able to live this journey and grow together with you.

<br>⚡ But this is not the end — it is only a new beginning. Ahead of us lie even more ideas, opportunities, and solutions. See you in the **UNIVER Project**! 🚀
